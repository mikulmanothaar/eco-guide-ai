# EcoGuide AI

EcoGuide AI is a Carbon Footprint Awareness Platform built using Flask.

## Features

- Carbon Footprint Calculator
- Carbon Grade (A-F)
- Emission Breakdown
- Pie Chart Visualization
- Personalized Recommendations

## Tech Stack

- Python
- Flask
- HTML/CSS
- Matplotlib
- Gunicorn

## Run Locally

```bash
pip install -r requirements.txt
python app.py
```

## Deployment

Hosted on Render.
